# Timer



Простой таймер с графическим интерфейсом на Qt.  

Мне хотелось разобраться, как работают графические приложения на C++.  

Я выбрала таймер, потому что в нём есть всё, что нужно для старта:

\- интерфейс

\- логика

\- обработка действий пользователя

\- обратная связь



Этот проект помог мне понять, как связать визуальную часть с кодом.



Интерфейс

\- Окно с LCD-дисплеем

\- Поле для ввода минут

\- Три кнопки: Старт, Пауза, Обнуление

\- Всё собрано в Qt Designer



Логика

\- При нажатии «Старт» начинается обратный отсчёт

\- «Пауза» останавливает таймер, не сбрасывая время

\- «Обнуление» сбрасывает таймер к текущему значению в поле ввода

\- По окончании таймер останавливается автоматически
